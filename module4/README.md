# Degen Token (ERC-20)

creating a ERC20 token and deploy it on the Avalanche network for Degen Gaming

## Description

-Minting new tokens
-Transferring tokens
-Redeeming tokens
-Checking token balance 
-Burning tokens

### Installing

* here you should be using remix ide 
* before that you should install metamask to your computer and must have avax token if you dont you can request at their website

### Executing program

1. deploy the contract
2. change the environment to web 3 provider and now you can check the transaction in snowtrace
